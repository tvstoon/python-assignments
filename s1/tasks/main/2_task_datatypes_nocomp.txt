
# تمرین ۱
# برای یک بازیکن متغیرهای زیر را بساز:
# نام بازیکن
# مقدار پول
# مقدار جان
# زنده بودن یا نبودن


# تمرین ۲
# برای یک ماشین متغیرهای زیر را بساز:
# نام ماشین
# سرعت ماشین
# مقدار بنزین
# قفل بودن یا نبودن

# سپس نوع داده همه را چاپ کن.


# تمرین ۳
# یک لیست برای وسایل بازیکن بساز.
# داخل آن نام اسلحه، تعداد گلوله و غذا قرار بده.
# سپس نوع داده لیست را چاپ کن.
